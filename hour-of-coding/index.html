Launched simple helper for work[pomodorro]

Feautures: - keeps track of your sessions, so you can kill it. When you start it will continue working from the same time. - has a beautiful timer - no dependencies

Install it like this: git clone https://github.com/VbifRkbvjd/pomodorro-cli Usage is like this bash python ./main.py Configuration is in config.py python SAVE_FILE = "pomodorro.db" WORK = 25 REST = 5 BIGREST = 25 ONE_SECOND = 1 If you want to make a pause. Close app by Ctrl+c. It will remember it last position when you start it again. Source goes here

One hour of coding.

During this contest, I indend to write something useful for about an hour. This blog engine was made exactly at one hour, and 23 minutes. During this time I:

  1. Thought out the design and concept of blog
  2. Studied jinja2 and flask frameworks
  3. Studied ways of calculation tag sizes in a cloud
  4. Implemented tagging system
  5. Implemented some beautify tricks with jquery
  6. Made design,and impleneted it
  7. Debugged the result.
  8. Figured out how to use Flask-static
  9. Deployed it to github

Estimated time is 83 minutes.

Here goes architectural overview

In settings I've got navigation struct, like this: { "hello.html": {"date":"2015.02.28","tags":{"Hello","Contacts"},"posted":True}, "firstBlogPost.html": {"date":"2015.02.28","tags":{"hour-of-coding", "python","jinja2"},"posted":True}, "testPost.html": {"date":"2015.02.28","tags":{"hour-of-coding", "python","jinja2"},"posted":True}, "howToUseMarkDown.html":{"date":"2015.02.28","tags":{"markdown-tutorial","notForPeople"}, "posted":True}, } Here key is a category, and set tags contains possible categories for this kind of blogpost.

{% filter markdown %} My beautiful markdown {% endfilter %}

It is done like this, to have the ability implementing something, that is not possible in markdown.

That is it. Very simple straightforward, and easy to deploy in a static environment.

This blogEngine was definitely not possible without typeplate,flask,jQuery,neat,bourbon and lots of other free libraries.
